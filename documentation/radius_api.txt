GET
/search/au/property/geo/radius
Search by Radius


The Search by Radius operation allows users to retrieve a list of properties that surround a central geographical point (latitude, longitude). The user can filter by various basic attributes and will be returned a sorted list that includes summary data for each property record. The default sort of the results is ascending by the property's distance from the center point.
