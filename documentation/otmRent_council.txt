GET
/search/au/property/councilArea/{councilAreaId}/otmForRent
Search OTM For Rent by Council Area


Use the Property Search OTM For Rent by Council Area service to get a list of properties for a specific council area that are on the market (OTM) for Rent. You can filter and sort the results based on multiple parameters. The response includes a list of properties with property summary and the OTM-related data specific to the property's campaign. The results in the response are sorted in ascending order based on the property's address.

activeCampaignOnly
(query)
Filter on the status of the OTM For Rent campaign. Although a property can be identified as being OTM for rent,there are scenarios when ads may still be displayed despite the property being given for rent. This flag will help identify those campaigns which are truely active (no for rent events against the campaign).
This is passed as either 'true' or 'false'.
When excluded, this will returns only active campaigns.
activeCampaignOnly = true -> return only active campaigns.
activeCampaignOnly= false -> return both active and inactive . i.e. all
Default value is true.

The date value must be in the format YYYYMMDD (ISO 8601 format without the hyphens)

date=-20160101, the response includes all the properties that have the latest advertisement date before January 1, 2016.
